## Points System

EnConvo Cloud uses a points system for AI operations. For detailed pricing per model, visit our [Cloud Plan page](https://www.enconvo.com/cloud-plan).

<Tip>
Use your own API key with Standard or Premium plans to bypass the points system entirely and only pay for actual usage through your provider.
</Tip>

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="Can I use EnConvo for free?">
    Yes! EnConvo offers a free plan with 10 uses per day. All features are available in the free plan.
  </Accordion>
  <Accordion title="What's the difference between Standard and Premium?">
    | | Standard | Premium |
    |---|----------|---------|
    | Price | $24.5 | $49.5 |
    | Bonus Points | 50,000 | 150,000 |
    | Updates | 1 Year | Lifetime |
    | Devices | 1 | 3 |
    
    Premium is better value if you plan to use EnConvo long-term or on multiple devices.
  </Accordion>
  <Accordion title="What's the difference between License and Cloud plans?">
    - **License (Standard/Premium)**: One-time purchase, requires your own API key for unlimited use
    - **Cloud Plans**: Monthly/yearly subscription, no API key needed, uses points system
    
    Choose License if you have your own API keys. Choose Cloud if you prefer a hassle-free experience.
  </Accordion>
  <Accordion title="Do I need my own API key?">
    - **Free/Cloud Plans**: EnConvo provides points, no API key needed
    - **Standard/Premium**: Recommended for unlimited usage, but optional
    
    Using your own API key gives you unlimited usage without point costs.
  </Accordion>
  <Accordion title="What happens when I run out of points?">
    You can either:
    - Wait for daily/monthly reset
    - Purchase additional points
    - Use your own API key
  </Accordion>
  <Accordion title="Can I upgrade my plan?">
    Yes, you can upgrade anytime. Contact support for assistance with plan changes.
  </Accordion>
  <Accordion title="Is there a student discount?">
    Yes! Students receive discounts. Email support@enconvo.com with your student ID to get your discount code.
  </Accordion>
</AccordionGroup>
