## Overview

Voice Input transforms your speech into text with high accuracy, allowing you to dictate text, issue commands, and interact with AI using your voice. It's perfect for hands-free operation, long-form dictation, and accessibility needs.

## Activating Voice Input

### Keyboard Shortcut
Press `⌥ V` (Option + V) to start voice input.

### Menu Bar
Click the EnConvo menu bar icon → Start Voice Input

### SmartBar
Open SmartBar and click the microphone icon.

## Features

### High-Accuracy Transcription
EnConvo uses advanced speech recognition for accurate transcription:
- Multiple language support
- Accent adaptation
- Technical vocabulary recognition
- Punctuation prediction

### Voice Commands
Beyond dictation, use voice for AI commands:
- "Hey EnConvo, summarize this page"
- "Translate selection to Spanish"
- "What's on my screen?"

### Live Closed Captions
Enable real-time captions for any audio:
- System audio transcription
- Meeting captions
- Video subtitle generation

## Using Voice Input

<Steps>
  <Step title="Activate">
    Press `⌥ V` or click the microphone icon
  </Step>
  <Step title="Speak">
    Speak clearly at a normal pace. A visual indicator shows EnConvo is listening.
  </Step>
  <Step title="Review">
    Your speech appears as text in real-time. Review for accuracy.
  </Step>
  <Step title="Process">
    Press Enter to send to AI, or continue speaking for more input.
  </Step>
</Steps>

## Voice Input Modes

### Dictation Mode
Pure speech-to-text conversion. Your words appear exactly as spoken.

```
You say: "Hello comma my name is John period"
Result: "Hello, my name is John."
```

### Command Mode
Voice commands are processed as AI queries.

```
You say: "Summarize this email"
Action: AI summarizes the current email
```

### Continuous Mode
Keep voice input active for extended dictation sessions.

## Voice Commands Reference

### Navigation Commands
| Command | Action |
|---------|--------|
| "Open SmartBar" | Opens SmartBar |
| "Show PopBar" | Activates PopBar |
| "Open settings" | Opens EnConvo settings |
| "New conversation" | Starts new chat |

### AI Commands
| Command | Action |
|---------|--------|
| "Summarize [this/screen]" | Summarizes content |
| "Translate to [language]" | Translates selected text |
| "Explain [this/selection]" | Explains content |
| "Write [type] about [topic]" | Generates content |

### Control Commands
| Command | Action |
|---------|--------|
| "Stop listening" | Ends voice input |
| "Cancel" | Cancels current input |
| "Delete that" | Removes last transcription |
| "Clear all" | Clears all input |

## Live Closed Captions

Enable real-time captions for any audio on your Mac:

<Steps>
  <Step title="Enable Captions">
    Go to Settings → Voice Input → Live Captions → Enable
  </Step>
  <Step title="Choose Audio Source">
    Select "System Audio" for all sounds or specific application audio
  </Step>
  <Step title="Position Captions">
    Drag the caption window to your preferred position
  </Step>
  <Step title="Customize Appearance">
    Adjust font size, background opacity, and text color
  </Step>
</Steps>

### Caption Features
- Real-time transcription
- Speaker identification (when possible)
- Adjustable display duration
- Export to text file

## Configuration

### Settings → Voice Input

| Setting | Description |
|---------|-------------|
| **Primary Language** | Main language for recognition |
| **Auto-detect Language** | Automatically switch languages |
| **Punctuation** | Auto-insert punctuation |
| **Profanity Filter** | Filter explicit content |
| **Activation Shortcut** | Customize `⌥ V` shortcut |
| **Listening Indicator** | Visual/audio feedback style |
| **Continuous Mode** | Stay active after processing |

## Tips for Better Recognition

<AccordionGroup>
  <Accordion title="Speak clearly">
    Maintain a consistent pace and enunciate clearly. Avoid mumbling or speaking too quickly.
  </Accordion>
  <Accordion title="Reduce background noise">
    Use in a quiet environment or use a directional microphone. Close windows and doors if needed.
  </Accordion>
  <Accordion title="Use voice commands for punctuation">
    Say "period", "comma", "question mark" for punctuation, or enable auto-punctuation.
  </Accordion>
  <Accordion title="Train your voice (optional)">
    Some models improve with use. Correct transcription errors to help the system learn your voice.
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Microphone not detected">
    1. Check System Settings → Privacy & Security → Microphone
    2. Ensure EnConvo has microphone permission
    3. Verify microphone is selected in EnConvo settings
  </Accordion>
  <Accordion title="Poor transcription accuracy">
    1. Check microphone quality and positioning
    2. Reduce background noise
    3. Try a different language/accent setting
    4. Speak more slowly and clearly
  </Accordion>
  <Accordion title="Voice commands not recognized">
    1. Ensure Command Mode is enabled
    2. Say commands clearly and completely
    3. Check that the command is supported
  </Accordion>
</AccordionGroup>
