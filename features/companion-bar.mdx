## Overview

The Companion Bar is a persistent sidebar that provides continuous AI assistance while you work. Unlike SmartBar which appears on demand, Companion Bar stays visible on the side of your screen for ongoing conversations and quick reference.

## Opening Companion Bar

- **Keyboard shortcut**: Press `⌥ C` (Option + C)
- **Click**: Click the EnConvo menu bar icon and select "Show Companion Bar"
- **SmartBar**: Type `/companion` in SmartBar

## Interface

The Companion Bar consists of several sections:

### Chat Area
The main conversation interface where you interact with AI. Messages are displayed in a familiar chat format with:
- Your messages on the right
- AI responses on the left
- Timestamps and model indicators
- Copy, regenerate, and edit buttons

### Context Panel
Shows what context is currently being used:
- Active application
- Current screen snapshot
- Referenced files
- Knowledge base sources

### Quick Actions
Frequently used actions pinned for easy access:
- Summarize screen
- Translate selection
- Explain code
- Custom actions

## Features

### Persistent Conversations

Unlike SmartBar, Companion Bar maintains conversation history:
- Continue previous conversations
- Reference earlier messages
- Build on context over time

### Multi-Model Support

Switch between AI models without losing context:

```
Available Models:
├── OpenAI GPT-4
├── OpenAI GPT-3.5
├── Claude 3 Opus
├── Claude 3 Sonnet
├── Gemini Pro
└── Local LLMs (Ollama)
```

### Screen Awareness

Companion Bar continuously monitors your screen:
- Automatic context updates
- Smart suggestions based on current work
- Proactive assistance options

### File Integration

Drag and drop files directly into Companion Bar:
- Documents (PDF, Word, etc.)
- Images for analysis
- Code files for review
- Any file type for processing

## Configuration

### Position
Choose where Companion Bar appears:
- **Right side** (default)
- **Left side**
- **Floating window**

### Size
Adjust the width:
- **Compact**: 300px
- **Standard**: 400px
- **Wide**: 500px

### Behavior
- **Auto-hide**: Collapse when not in use
- **Always on top**: Stay visible over other windows
- **Follow focus**: Move to current display

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `⌥ C` | Toggle Companion Bar |
| `⌘ Enter` | Send message |
| `⌘ N` | New conversation |
| `⌘ ↑` | Previous conversation |
| `⌘ ↓` | Next conversation |
| `⌘ K` | Clear conversation |

## Companion Bar vs SmartBar

| Feature | SmartBar | Companion Bar |
|---------|----------|---------------|
| Activation | On-demand | Persistent |
| Position | Center overlay | Side panel |
| History | Single query | Full conversation |
| Use case | Quick tasks | Extended work |
| Screen space | Minimal | Dedicated area |

<Tip>
Use SmartBar for quick, one-off queries and Companion Bar for extended work sessions where you need ongoing AI assistance.
</Tip>
