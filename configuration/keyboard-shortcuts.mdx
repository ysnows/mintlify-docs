## Overview

EnConvo uses keyboard shortcuts to give you fast access to AI features from anywhere on your Mac. All shortcuts work globally -- you do not need to have EnConvo in the foreground.

<Note>
All default shortcuts listed here can be customized in Settings. If a shortcut conflicts with another application, EnConvo will notify you and suggest alternatives.
</Note>

## Global Shortcuts

These shortcuts work from any application on your Mac.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Shift + Cmd + D` | Open SmartBar | Launch the Spotlight-like AI interface |
| `Shift + Cmd + Space` | Open PopBar | Show the floating AI toolbar on selected text |
| `Shift + Cmd + E` | Open Companion Bar | Open the sidebar AI companion |
| `Shift + Cmd + M` | Toggle Dictation | Start/stop voice dictation |
| `Shift + Cmd + S` | Screenshot OCR | Capture screen region and extract text |
| `Shift + Cmd + T` | Screenshot Translate | Capture screen region, OCR, and translate |

<Tip>
If you only remember one shortcut, make it `Shift + Cmd + D` for SmartBar. From there, you can access every EnConvo feature by typing commands.
</Tip>

## SmartBar Shortcuts

These shortcuts are available when the SmartBar is open.

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message / Execute command |
| `Shift + Enter` | New line in input |
| `Cmd + N` | New conversation |
| `Cmd + P` | Pin/Unpin SmartBar window |
| `Cmd + W` | Close SmartBar |
| `Escape` | Close SmartBar |
| `Up Arrow` | Navigate to previous message in history |
| `Down Arrow` | Navigate to next message in history |
| `Cmd + K` | Clear conversation |
| `Cmd + L` | Focus input field |

### Input Modifiers

| Shortcut | Action |
|----------|--------|
| `@` | Show available tools and plugins |
| `#` | Add context (screen, browser, clipboard, etc.) |
| `/` | Show available skills and commands |
| `Cmd + V` | Paste text or image from clipboard |

## Conversation Shortcuts

Available when viewing a conversation in the sidebar or SmartBar.

| Shortcut | Action |
|----------|--------|

| `Cmd + Enter` | Regenerate last response |
| `Cmd + Shift + E` | Export conversation |
| `Cmd + Shift + S` | Save conversation as markdown |

## PopBar Shortcuts

Available when PopBar is displayed on selected text.

| Shortcut | Action |
|----------|--------|

| `Enter` | Execute the selected action |
| `Tab` | Cycle through available actions |
| `Cmd + C` | Copy result to clipboard |
| `Cmd + V` | Replace selected text with result |

## Extension-Specific Shortcuts

Individual extensions can define their own hotkeys. These are configured per-extension in Settings.

### Setting an Extension Hotkey

<Steps>
  <Step title="Open Settings">
    Press `Cmd + ,` or go to Settings
  </Step>
  <Step title="Navigate to the extension">
    Find the extension in the Extensions list
  </Step>
  <Step title="Find the Hotkey preference">
    Look for preferences with type "hotkey" in the command settings
  </Step>
  <Step title="Record your shortcut">
    Click the hotkey field and press your desired key combination
  </Step>
  <Step title="Save">
    The shortcut is active immediately
  </Step>
</Steps>

### Common Extension Hotkeys

Many users configure these popular shortcuts:

| Extension | Suggested Shortcut | Action |
|-----------|-------------------|--------|
| Translate | `Cmd + Shift + T` | Translate selected text |
| OCR | `Cmd + Shift + O` | Screenshot OCR |
| Fix Spelling | `Cmd + Shift + G` | Fix grammar on selected text |
| Read Aloud | `Cmd + Shift + R` | Read selected text aloud |
| Web Search | `Cmd + Shift + W` | Search the web for selected text |

## Customizing Shortcuts

### Changing Global Shortcuts

<Steps>
  <Step title="Open Settings">
    Press `Cmd + ,`
  </Step>
  <Step title="Go to the feature section">
    Navigate to SmartBar, PopBar, or Companion Bar settings
  </Step>
  <Step title="Click the shortcut field">
    Click the current shortcut display
  </Step>
  <Step title="Record new shortcut">
    Press your desired key combination. The new shortcut is saved automatically.
  </Step>
</Steps>

### Modifier Key Options

EnConvo supports these modifier keys in shortcut combinations:

| Symbol | Key |
|--------|-----|
| `Cmd` | Command |
| `Shift` | Shift |
| `Opt` | Option/Alt |
| `Ctrl` | Control |
| `Fn` | Function |

Shortcuts must include at least one modifier key (Cmd, Shift, Opt, or Ctrl) combined with a regular key.

### Modifier Flag Actions

Some commands support modifier flag actions -- hold a modifier key when pressing Enter to perform an alternative action on the result:

| Modifier | Default Action |
|----------|---------------|
| `Cmd + Enter` | Copy result to clipboard |
| `Shift + Enter` | Paste result into frontmost app |
| `Opt + Enter` | Insert result below cursor |

Configure modifier flag actions in each command's preferences.

## Resolving Shortcut Conflicts

If a shortcut conflicts with another application or macOS system shortcut:

<Steps>
  <Step title="Check for conflicts">
    EnConvo shows a warning icon next to conflicting shortcuts in Settings
  </Step>
  <Step title="Identify the conflict">
    Common conflicts include:
    - macOS Spotlight (`Cmd + Space`)
    - macOS screenshots (`Cmd + Shift + 3/4/5`)
    - Third-party apps like Alfred, Raycast, or Karabiner
  </Step>
  <Step title="Change the conflicting shortcut">
    Either change the EnConvo shortcut or disable the conflicting shortcut in the other application
  </Step>
  <Step title="System shortcut conflicts">
    For macOS system shortcuts, go to System Settings → Keyboard → Keyboard Shortcuts to disable or remap them
  </Step>
</Steps>

<Warning>
Avoid using shortcuts already assigned to macOS system functions like `Cmd + Space` (Spotlight), `Cmd + Tab` (App Switcher), or `Cmd + Q` (Quit). These cannot be reliably overridden.
</Warning>
