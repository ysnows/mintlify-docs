## Quick Start

<Steps>
  <Step title="Select Text">
    Highlight the text you want to translate in any application.
  </Step>
  <Step title="Trigger Translation">
    Use the hotkey `Option + D` (default) or invoke the **Translator** command from PopBar or SmartBar.
  </Step>
  <Step title="View Translation">
    The translation appears instantly. EnConvo automatically detects the source language and translates to your configured target language.
  </Step>
</Steps>

## Translation Modes

### Sentence and Paragraph Translation

When you provide full sentences or paragraphs, EnConvo's AI translation:

1. Translates the entire text into the target language
2. Preserves domain-specific terms (e.g., "LLM", "NASA") in their original form
3. Provides a word-by-word grammatical breakdown including:
   - Source and target language mapping
   - Grammatical function (Subject, Object, etc.)
   - Detailed explanations

### Single Word / Phrase Translation

When you provide a single word or short phrase, EnConvo activates dictionary mode:

1. **Translation and Pronunciation** -- IPA for European languages, pinyin for Mandarin
2. **Extended Translation** -- all possible translations by word attribute (noun, verb, adjective, etc.) with example sentences
3. **Source Language Interpretation** -- paraphrased definition in the original language for nuance

**Example output for "snow":**

```
snow /snoʊ/; "雪" Pinyin [xuě]

Extended Translation:
* [n]: 雪 (xuě): frozen precipitation, e.g., "The snow is falling." = "雪在下。"
* [v]: 下雪 (xià xuě): to snow, e.g., "It is snowing outside." = "外面在下雪。"
* [adj.]: 雪白的 (xuě bái de): snow-white, e.g., "a snow-white coat" = "雪白的外套"
```

## Configuration

### Language Settings

<Steps>
  <Step title="Set Source Language">
    Open the Translator command settings and configure **Source Language**. Set to **Auto** for automatic detection, or choose a specific language.
  </Step>
  <Step title="Set Target Language">
    Configure the **Target Language** -- the language you want to translate into. The default is Japanese; change it to your preferred language.
  </Step>
</Steps>

EnConvo supports all ISO 639-1 language codes. The language list is comprehensive and includes all major world languages.

<Tip>
Automatic language detection works well for most cases. Only set a specific source language if you regularly translate from one particular language and want to avoid detection delays.
</Tip>

### Translation Provider Setup

<Tabs>

    2. Under **AI Model**, select your preferred LLM provider

    **Advantages:** Understands context, handles idioms, provides explanations
  </Tab>
  <Tab title="DeepL">

## Features

### Screenshot Translation (OCR + Translate)

Translate text directly from your screen:

1. Trigger the **Screenshot Translate** command
2. Select the screen area containing the text
3. EnConvo performs OCR to extract the text, then translates it

This is perfect for translating:
- Foreign language websites
- Images containing text
- Non-selectable text in applications
- PDF documents displayed on screen

### Hotkey Translation

The fastest way to translate is with a keyboard shortcut:

| Hotkey | Action |
|--------|--------|
| `Option + D` | Translate selected text (default) |

You can customize this hotkey in the Translator command preferences.

### Modifier Key Actions

Configure additional actions triggered by holding modifier keys when translating from PopBar:

| Modifier | Default Action |
|----------|----------------|
| `Option` | Copy translation to clipboard |
| `Shift` | Paste translation directly |

These can be customized in the **Modifier Flags Actions** settings.

### Post-Translation Actions

Configure what happens automatically after translation:

| Action | Description |
|--------|-------------|
| **None** | Just display the translation (default) |
| **Paste** | Automatically paste the translation at your cursor |
| **Insert Below** | Insert the translation below the original text |
| **Copy** | Copy the translation to clipboard |

### Text-to-Speech Integration

Have translations read aloud automatically:

1. Open Translator settings
2. Find the **Text-to-Speech** group
3. Enable **Automatically Play TTS Audio**
4. Select your preferred TTS provider

This is especially useful for language learning -- see and hear the translation simultaneously.

## Tips for Better Translations

<AccordionGroup>
  <Accordion title="Choose the right provider for your language pair">
    - **European languages:** DeepL generally produces the most natural translations
    - **Chinese/Japanese/Korean:** Caiyun or Youdao specialize in CJK languages
    - **Technical content:** AI/LLM translation understands context better
    - **General purpose:** Google Translate has the broadest language coverage
  </Accordion>
  <Accordion title="Provide complete sentences">
    Translation accuracy improves with more context. Complete sentences translate better than fragments. If translating technical content, include surrounding sentences for context.
  </Accordion>
  <Accordion title="Use AI translation for ambiguous text">
    The LLM-powered translator handles ambiguity, idioms, and context-dependent meanings better than traditional engines. It also explains its translation choices.
  </Accordion>
  <Accordion title="Set the target language explicitly">
    While auto-detection works for the source language, always explicitly set your target language to avoid unexpected results.
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Translation hotkey not working">
    1. Check that the hotkey `Option + D` is not conflicting with another application
    2. Verify that EnConvo has accessibility permissions in System Settings
    3. Ensure text is selected before pressing the hotkey
    4. Try reconfiguring the hotkey in Translator settings
  </Accordion>
  <Accordion title="Wrong target language">
    1. Open Translator settings
    2. Verify the **Target Language** is set correctly
    3. If using Auto for source language, ensure it is detecting correctly
  </Accordion>
  <Accordion title="DeepL translation fails">
    1. Check your DeepL API key is valid and has remaining quota
    2. If using the Free tier, ensure you have not exceeded the monthly limit
    3. Try switching to Google Translate or AI translation as a fallback
  </Accordion>
  <Accordion title="Screenshot translate shows garbled text">
    1. Ensure the selected area is clear and high resolution
    2. Check that the correct OCR provider is configured
    3. Try selecting a smaller, more focused area
    4. The OCR provider may not support the language in the image
  </Accordion>
</AccordionGroup>
